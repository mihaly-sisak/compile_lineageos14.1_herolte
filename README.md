# Compile LineageOS 14.1 for herolte

 - Clone android repo, follow guide https://wiki.lineageos.org/devices/herolte/build/
 - Get binary blobs from https://github.com/TheMuppets/proprietary_vendor_samsung/tree/cm-14.1/hero-common
 - Patch `lineage/kernel/samsung/universal8890`
 - Run `start.sh`
 - Flash output, follow guide https://wiki.lineageos.org/devices/herolte/install/
